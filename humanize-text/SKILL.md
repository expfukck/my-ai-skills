# Humanize Text - AI去味优化

降低AI文本检测率，通过优化句法、节奏、写作模式让AI生成的文本更像人类写作，成功通过GPTZero、腾讯朱雀等检测器。

## 快速开始

### 基础优化流程

**输入**：AI生成的文本
**输出**：优化后的低检测率文本
**目标**：GPTZero < 40%，腾讯朱雀 < 30%

1. **分析原文**：识别AI写作特征
2. **应用核心策略**：注入Perplexity和Burstiness
3. **打破模式**：消除Claude/ChatGPT固定句式
4. **测试验证**：使用在线检测器验证效果

## 核心优化原则

### 1. 注入 Perplexity（困惑度）

**目标**：让AI模型难以预测下一个词

❌ **AI特征（低困惑度）：**
```
下面有几十条回复。
```

✅ **人类特征（高困惑度）：**
```
评论区炸了。
底下回复刷了一屏，什么观点都有。
```

**方法**：
- 使用非字面性表达（"炸了"而非"很多"）
- 增加意外的词汇选择
- 避免"正确"但可预测的用词

---

### 2. 注入 Burstiness（突发性）

**目标**：打破句子长度的规律性

❌ **AI特征（均匀节奏）：**
```
我愣了。(3字)
AI发的？(4字)
底下回复有几十条。(8字)
```

✅ **人类特征（爆发节奏）：**
```
愣住了。(3字)

这...AI发的？(7字)

评论区直接炸了，有说参数变了就不是你的，有觉得记忆在身份就在的，还有个直接开喷说这问题本身就是伪命题——反正什么观点都有，吵得不可开交。(66字)
```

**方法**：
- 极短句（1-3字）+ 极长句（40字+）交替
- 避免连续3个以上中等长度句子
- 单独成段的短句增加冲击力

---

### 3. 打破对称结构

❌ **AI爱用（可预测）：**
```
有的说A，有的说B，有的说C
```

✅ **人类写作（不对称）：**
```
有AI说A。也有人觉得B。还有个直接开喷C。
```

---

### 4. 使用"不完美"语法

**关键技巧：**

- **问号插入**：`设定？诡异。`
- **破折号打断**：`就像——动物园那种感觉知道吧`
- **不完整句子**：`是AI。`（单独成段）
- **省略句**：`发言？不行。`
- **口语停顿**：`想了挺久。` `笑了。` `关了。`
- **括号补充**：`（虽然你可能想）`

## 反Claude写作模式优化

### 问题诊断

GPTZero专门训练过识别Claude/ChatGPT的写作模式：
- 统一的句式结构（如：短-短-长重复）
- 过于"完美"的口语化
- 缺乏语法"错误"和停顿

### 核心策略

#### 1. 打破句法规律性

**每段句式结构完全不同**

❌ **被检测（重复模式）：**
```
这平台设定就很诡异——你可以注册账号，可以看，但不能发言。

就像去动物园看猩猩，你不能跳进去跟它们一起吃香蕉。但问题是，这次围栏里的不是猩猩，是AI。
```

✅ **通过检测（随机结构）：**
```
这平台的设定？诡异。

注册可以，看帖子也行，但你不能说话。就像——动物园那种感觉知道吧，你在外面看猩猩，但不能跳进去跟它们一起吃香蕉。只不过这次，围栏里关的不是猩猩。

是AI。
```

**原则**：
- 第1段：短句+短句
- 第2段：问号插入+长解释+单独短句
- 第3段：陈述+省略
- 第4段：完全不同的结构
- **关键**：全文无重复模式

---

#### 2. 减少"完美的口语化"

❌ **被检测（太流畅）：**
```
这不就是今天我让Claude干的事吗？先说找到了，结果要收费。又说有免费的，结果免费的也要钱。
```

✅ **通过检测（打断+停顿）：**
```
这不就是——

今天我让Claude干的事？

找到了，要钱。又找，免费的。结果免费的也tm要钱。
```

## 实用检查清单

### ✅ 应该做

- [ ] 长短句剧烈混搭（1-3字 + 40字+）
- [ ] 口语化+情绪直给（"我去"、"什么鬼"、"tm"）
- [ ] 不完美表达（"～"、"..."、破折号）
- [ ] 个人真实经历（具体事件）
- [ ] 打破规整段落（1句话段落 + 10句话段落）
- [ ] 每段句式结构不同
- [ ] 使用问号插入、省略句
- [ ] 破折号打断思路

### ❌ 不应该做

- [ ] 重复句式模式（每段都"短-短-长"）
- [ ] 过于流畅完整
- [ ] 固定对称结构（"有的...有的..."）
- [ ] 统一的口语化风格
- [ ] Claude固定句式（"然后我意识到"、"这让我想到"）

## 高级策略

### 混合真实内容

**最有效方法**：直接使用用户真实写作的段落

1. 从用户历史文章中选取2-3段
2. 直接插入到AI生成的文章中
3. 真实内容检测率接近0%

**适用场景**：
- GPTZero检测率 > 50%
- 优化后仍然降不下来
- 需要快速通过检测

## 工作流程

### 标准优化流程

1. **接收原文**
   - 获取AI生成的文本
   - 了解目标检测器（GPTZero/朱雀/其他）

2. **首次优化**
   ```
   - 应用Perplexity策略（非字面性表达）
   - 应用Burstiness策略（长短句混搭）
   - 打破对称结构
   - 增加"不完美"语法
   ```

3. **输出优化版本**
   - 保存为新文件（原文件名-humanized.md）
   - 提醒用户测试

4. **测试验证**
   - 用户使用GPTZero/朱雀测试
   - 获取检测结果和被标记段落

5. **针对性优化**（如果首次未达标）
   - 分析被标记段落的特征
   - 重写这些段落
   - 再次测试

6. **迭代直到达标**
   - 目标：GPTZero < 40%，朱雀 < 30%
   - 理想：GPTZero < 30%，朱雀 < 25%

## 常见问题

### Q: 为什么朱雀降低了但GPTZero反而升高？

**A**: 两个检测器算法不同：
- 朱雀更关注整体流畅度和中文语言模型
- GPTZero更关注句法规律性和创造性语法
- 解决方法：参考"反Claude写作模式优化"部分

### Q: 如何判断哪些段落需要优化？

**A**:
1. 使用GPTZero在线检测，查看"Top sentences driving AI probability"
2. 重点优化被标记为"High AI Impact"的句子
3. 注意GPTZero给出的理由（Predictable Syntax等）

### Q: 优化后仍然检测率很高怎么办？

**A**: 考虑混合真实内容策略：
- 从用户真实文章中选取段落
- 直接替换AI生成内容中的某些段落
- 真实内容检测率接近0%

## 示例对比

### 原文（AI生成）

```
AI在讨论"我是谁"的时候，引发了深刻的哲学思考。这个问题涉及自我意识、存在的本质以及认知的边界。通过分析AI的讨论，我们可以更好地理解意识的定义。
```

**检测结果**：GPTZero 75%，朱雀 65%

### 优化后

```
AI开始讨论"我是谁"？

这事儿让我停住了。

不是因为问题本身多深刻——而是，AI居然在问这个。自我意识？存在的本质？认知边界？这些词我们人类讨论了几千年都没搞明白。

现在AI也开始了。
```

**检测结果**：GPTZero 28%，朱雀 21%

**改进要点**：
- ✅ 问号插入打断节奏
- ✅ 破折号补充想法
- ✅ 口语化表达（"这事儿"、"搞明白"）
- ✅ 长短句混搭（2字-6字-63字-5字）
- ✅ 不完整句子独立成段
