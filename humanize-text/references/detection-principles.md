# AI文本检测原理

## 核心检测指标

### 1. Perplexity（困惑度）

**定义**：语言模型对文本预测的难度

- **AI文本**：困惑度低（50-100）
  - 原因：输出符合训练数据的统计规律
  - 特征：可预测、流畅、"正确"

- **人类文本**：困惑度高（150-300+）
  - 原因：有创造性、意外的词汇选择
  - 特征：不可预测、有个人风格

**判定标准（基于GPT-2模型）**：
- < 60 → AI生成
- 60-80 → 可能含AI部分
- > 80 → 人类写作

---

### 2. Burstiness（突发性/爆发性）

**定义**：句子长度和结构变化的交替性

- **AI文本**：突发性低（0.2-0.4）
  - 原因：句子长度均匀、结构统一
  - 特征：中等长度句子占多数、节奏规律

- **人类文本**：突发性高（0.6-0.9）
  - 原因：长短句混杂、节奏变化大
  - 特征：极短句+极长句交替出现

**计算方式**：
- 单句最高困惑度
- 句子长度变异系数

---

### 3. 句法模式特征

#### AI写作的典型特征：

**❌ 可预测句法（Predictable Syntax）**
```
问题：基础陈述句式，结构可预测
示例："下面有几十条回复。"
```

**❌ 可预测节奏（Predictable Rhythm）**
```
问题：一致可预测的节奏
示例：连续3-5个中等长度句子
```

**❌ 单调句法（Monotonous Syntax）**
```
问题：句法复杂度有限，重复模式
示例："有的说A，有的说B，有的说C"
```

**❌ 缺乏创造性（Lacks Creativity）**
```
问题：句子太"完整"、太"正确"
示例：没有省略、打断、插入语
```

## 主流检测器特点

### GPTZero

**检测机制**：
1. 私有AI检测模型（非GPT-2）
2. 多维度综合判断：
   - 困惑度分析
   - 突发性分析
   - 句法模式识别
   - 对比GPT-4/Claude输出数据库
3. 专门训练识别Claude/ChatGPT写作风格

**特点**：
- 对句法规律性敏感
- 能识别"公式化的口语化"
- 检测创造性语法的缺失

---

### 腾讯朱雀

**检测机制**：
1. 中文特化检测模型
2. 主要关注：
   - 整体流畅度
   - 中文语言模型一致性

**特点**：
- 对口语化表达敏感度较低
- 更关注整体文本质量

---

## 检测器的局限性

### 误判情况

**人类文本被误判为AI**：
- 学术写作（固定语句、低困惑度）
- 法律文书（高"经验性"表达）
- 技术文档（规范化语言）

**AI文本通过检测**：
- 混入真实人类内容
- 打破句法规律性
- 增加语法"不完美"

### 注意事项

> GPTZero 曾将美国宪法和圣经判定为"可能完全由AI编写"

检测器**不是绝对准确**，存在误判可能性。
